AI Chat Application(Integretion of ChatGPT)

* Designed RESTful APIs to handle real-time chat requests and responses.
* Stored user and assistant messages in MongoDB to maintain conversational context.
* Used UUID v4 to uniquely identify chat sessions across multiple users.
* Tested APIs using Postman and handled errors with proper validation.
* Converted UTC timestamps to IST for user-friendly display.

Technologies Used:
Node JS || Express JS || MongoDB || OpenAI API
